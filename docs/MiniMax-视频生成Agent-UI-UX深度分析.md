## 四、动效与过渡

### 4.1 全局过渡配置

```css
:root {
  --transition-fast: 150ms ease;
  --transition-default: 200ms ease;
  --transition-slow: 300ms ease;
  --ease-out-expo: cubic-bezier(0.16, 1, 0.3, 1);
}
```

### 4.2 组件动效

| 组件 | 属性 | 时长 | 缓动 |
|------|------|------|------|
| 按钮 Hover | transform, box-shadow | 150ms | ease |
| 卡片 Hover | transform, border-color, box-shadow | 200ms | ease-out |
| Tab 切换 | background, border-color | 150ms | ease |
| 折叠面板 | height, opacity | 300ms | ease-out-expo |
| 页面加载 | opacity | 200ms | ease |
| Modal 弹入 | opacity, transform | 300ms | ease-out |
| 列表项进入 | opacity, translateY | 200ms | ease-out, staggered 50ms |

### 4.3 微交互细节

**卡片 Hover**：
```css
.card:hover {
  transform: translateY(-2px);
  border-color: transparent;
  background: linear-gradient(#1a1a1a, #1a1a1a) padding-box,
              linear-gradient(135deg, #8B5CF6, #3B82F6) border-box;
  box-shadow: 0 8px 32px rgba(139, 92, 246, 0.15);
}
```

**按钮 Click**：
```css
.button:active {
  transform: scale(0.98);
}
```

**输入框 Focus**：
```css
.input:focus {
  outline: none;
  border-color: #8B5CF6;
  box-shadow: 0 0 0 3px rgba(139, 92, 246, 0.2);
}
```

**侧边栏分类 Hover**：
```css
.sidebar-item:hover {
  background: rgba(139, 92, 246, 0.1);
```

## 六、交互逻辑深度解析

### 6.1 API Key 校验流程

```
┌─────────────────────────────────────────┐
│         用户点击"使用"按钮              │
└─────────────────┬───────────────────────┘
                  ▼
        ┌─────────┴─────────┐
        │  检查 API Key      │
        ▼                   ▼
   ┌─────────┐         ┌─────────┐
   │ 未配置   │         │  已配置   │
   └────┬────┘         └────┬────┘
        ▼                   ▼
   ┌─────────┐         ┌─────────┐
   │ 触发警告  │         │ 继续流程  │
   │ 引导配置  │         │          │
   └─────────┘         └────┬────┘
                             ▼
                    ┌─────────────────┐
                    │ 检查必填参数     │
                    └────────┬────────┘
                             ▼
                    ┌─────────────────┐
                    │ 调用生成 API    │
                    └─────────────────┘
```

### 6.2 分类切换逻辑

```
┌─────────────────────────────────────────────────────┐
│  用户点击分类 Tab                                    │
└─────────────────────┬───────────────────────────────┘
                      ▼
┌─────────────────────────────────────────────────────┐
│  1. 更新 URL 参数 (?category=xxx) - 可选             │
│  2. 更新 Tab Bar 选中态                             │
│  3. 侧边栏对应分类高亮                              │
│  4. 卡片区域显示 Loading 骨架屏（可选）              │
│  5. 执行过滤逻辑                                    │
│  6. 动画过渡旧列表 → 新列表                         │
└─────────────────────────────────────────────────────┘
```

**过渡动画**：
- 旧卡片：opacity 1→0, translateY 0→-10px, 150ms
- 新卡片：opacity 0→1, translateY 10px→0, 200ms, staggered 30ms

```css
@keyframes skeleton-loading {
  0% { background-position: 200% 0; }
```
