# 2026-04-25 Chat Model Series Badge Plan

## 目标
- 让聊天模型下拉中的 `GPT-5.x`、`GPT-4.1`、`GPT-4o`、`o Series` 等系列区分更显式，降低用户辨识成本。

## 范围
- `public\js\app.js`
- `public\css\style.css`
- 与聊天模型下拉展示直接相关的测试文件

## 假设
- 当前问题不在模型数据缺失，而在 UI 呈现层没有把“系列身份”和“能力标签”分层显示。
- 保留现有分组逻辑与能力标签逻辑，只补充最小的显式系列徽标。

## 风险
- 如果系列徽标和能力标签视觉权重接近，会让信息层级变乱。
- 如果增加的徽标宽度控制不好，可能影响下拉选项在窄宽度下的换行与高度节奏。
