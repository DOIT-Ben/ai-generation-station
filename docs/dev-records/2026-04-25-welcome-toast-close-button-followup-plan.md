# 2026-04-25 登录欢迎浮层关闭按钮续处理计划

## 目标
为登录成功后的欢迎浮层补上右上角关闭按钮，让它更接近参考图，同时保证用户手动关闭后仍能顺畅跳转。

## 范围
- `public\js\site-shell.js`
- `public\css\style.css`
- 如有必要，相关聚焦测试

## 假设
- 本轮只处理欢迎浮层关闭按钮，不调整登录成功后的整体欢迎风格。
- 用户点击关闭按钮后，应立即结束浮层并继续后续跳转，而不是中断登录流程。

## 风险
- 如果关闭按钮只移除 DOM 不处理 Promise，可能导致登录跳转卡住。
- 如果按钮可点击区域太小，交互体验会很差。
