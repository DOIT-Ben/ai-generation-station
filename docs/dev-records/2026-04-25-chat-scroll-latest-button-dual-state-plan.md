# 2026-04-25 Chat Scroll Latest Button Dual State Plan

## 目标
- 将“回到最新回复”按钮做成双状态表达：普通状态更克制，有新回复时更强调。

## 范围
- `public\index.html`
- `public\js\app.js`
- `public\css\style.css`
- 与按钮状态样式直接相关的最小测试

## 假设
- 当前按钮基础可用，下一步优化重点是状态表达而不是交互逻辑。
- “普通可返回最新”与“有新回复到达”应在视觉权重上区分。

## 风险
- 若强调状态过强，会压过聊天正文。
- 若状态切换只改文案不改视觉，用户感知收益会很有限。
