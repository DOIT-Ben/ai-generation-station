# 2026-04-25 Chat Scroll Latest Button Fix Plan

## 目标
- 修复聊天区“回到最新回复”按钮在浅色与护眼主题下看不清、文案显示不全的问题。

## 范围
- `public\css\style.css`
- 与该按钮样式直接相关的最小测试文件

## 假设
- 当前问题主要来自按钮对比度不足，以及单行不换行导致的文案裁切风险。
- 不修改聊天交互逻辑，只修正按钮样式与显示约束。

## 风险
- 如果按钮尺寸放大过度，可能遮挡聊天内容或输入区。
- 如果直接取消宽度约束，移动端可能发生横向溢出。
