# 护眼草稿纸主题

## 目标
- 为网站新增一个可切换的护眼模式。
- 视觉方向为淡黄色、草稿纸、纸张纹理感。
- 复用现有右上角主题切换入口，避免新增复杂设置面板。

## 范围
- 修改主站主题切换按钮、主题切换逻辑和 CSS 主题变量。
- 增加 `data-theme="paper"` 样式。
- 不修改后端接口、业务功能、导航结构和账号体系。

## 假设
- 当前主题切换为深色、浅色两态，用户需要在现有入口中继续切换。
- 护眼模式应比浅色模式更暖、更低对比、更接近草稿纸。
- 现有浅色主题仍需保留。

## 风险
- 三态主题切换若未处理历史偏好，可能读取到异常主题值。
- 只改变量不补局部组件样式，部分卡片或聊天区可能仍偏白或偏蓝。
- 草稿纸纹理过重会影响内容阅读，因此纹理需要轻量。

- 点击右上角主题按钮可在深色、浅色、护眼模式间循环。
- 进入护眼模式时，页面背景、侧栏、卡片、输入框、聊天区呈现淡黄色纸张风格。
- 主题偏好保存后再次加载仍能应用护眼模式。
- 现有深色、浅色主题不被移除。
