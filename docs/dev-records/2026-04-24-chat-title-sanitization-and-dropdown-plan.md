# 会话标题 Markdown 清理与模型下拉框优化计划

## 目标
- 清理聊天页会话标题与左侧会话预览中的 `##` 等 Markdown 残留。
- 优化模型下拉框的显示宽度与整体观感，避免文案显示不全。

## 范围
- 仅处理聊天页前端标题预览逻辑与模型下拉样式。
- 修改范围限定在：
  - `public\js\app.js`
  - `public\css\style.css`
- 不改后端、不改聊天接口、不改其他功能页。

## 假设
- 当前 `##` 出现在 `chat-conversation-title` 和左侧预览，是会话标题/摘要缺少 Markdown 清洗导致的。
- 当前模型下拉框的问题主要是：
  - 宽度偏窄
  - 文案缺少省略控制
  - 视觉细节不够完整

## 风险
- 标题清洗规则如果过重，可能误伤合法文本。
- 下拉框改宽后需要保持左上角浮层位置稳定。
