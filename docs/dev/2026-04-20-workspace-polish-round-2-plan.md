# 2026-04-20 Workspace Polish Round 2 Plan

## 目标
- 在已完成聊天流式、模板增强和额度折叠的基础上，继续把主工作台打磨成更适合长期日常使用的版本。
- 重点提升四块体验：模板区视觉与交互、导航栏瘦身、聊天阅读排版、会话列表管理感。
- 不改后端接口契约，优先通过前端布局、状态和交互补强产品感。

## 设计方向
- 风格方向：专业工作台，少一点装饰性卡片，多一点秩序和密度控制。
- 交互方向：重要操作就近可达，模板更像工具箱，会话更像正式产品中的对象列表。
- 视觉方向：侧栏更轻，主内容更突出，阅读区更舒展。

## 顺序化 TODO
1. 审查模板区、导航、聊天区、会话列表的当前结构与样式边界。
2. 升级模板区：增加搜索、模板预览摘要、统计信息和更完整的卡片层级。
3. 瘦身导航栏：压缩侧栏宽度、导航项体积和底部占用，提升主内容空间。
4. 精修聊天阅读：优化段落宽度、标题层级、列表与引用排版，提升长回复可读性。
5. 提升会话列表管理感：增加行内快捷操作和更清楚的状态标识。
6. 执行语法检查、静态断言和前端烟测，记录结果。

## 验收标准
- 模板区支持快速搜索，模板卡能预判内容而不是只看标题。
- 侧栏明显更瘦，工作台主区获得更多横向空间。
- AI 长回复阅读更顺，不像工具调试面板。
- 会话列表支持更就近的管理动作，像正式产品中的对象列表。
