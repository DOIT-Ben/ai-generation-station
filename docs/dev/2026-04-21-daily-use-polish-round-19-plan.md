# 2026-04-21 Daily Use Polish Round 19 Plan

## 目标
- 打磨聊天摘录与底部资产区的整理体验，解决“资产越来越多后不好清理、不好筛”的问题。
- 继续保持同页闭环，不新增独立资产页，不扩展后端。

## 设计结论
- 方案选择：在现有摘录管理面板内增加轻归档、筛选和批量操作。
- 不选独立资产管理页：当前阶段成本高、打断工作流。
- 不选复杂标签系统：超出当前打磨目标，先把归档和清理做好。

5. 打磨样式与移动端布局，保证新增操作不显乱。
6. 补齐文档和静态断言。
