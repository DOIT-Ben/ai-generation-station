- 已全量扩写 `chat` 模板：补齐角色、受众、目标、结构、风格和输出格式，提升一键开聊的首轮命中率。

- 已全量扩写 `cover` 模板：补齐构图、留白、灯光、材质、文字区域和视觉禁忌，提升图像生成可控性。
- 已全量扩写 `speech` 模板：把单句示例升级成更像真实播报稿、课程开场稿、情绪口播稿的完整文本。

#### 单任务复盘
- 新问题：聊天、歌词、音乐、封面、语音、翻唱六类模板的“丰富”定义并不相同，必须按场景定制，而不是统一加长。
