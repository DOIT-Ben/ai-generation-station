## 一、页面概览

**页面定位**：MiniMax B端解决方案 - 视频生成工具后台

**核心功能**：提供预设视频模板，用户选择模板后快速生成专业视频

### 3.4 右侧主内容区

#### 3.4.1 模板分类 Tab（顶部）

| 元素 | 描述 |
|------|------|
| Tab 容器 | 横向滚动（如果分类多） |
| Tab 项 | 分类图标 + 文字 |

**Tab 列表**：
- 全部

- 写真风格
- 文生视频
- 电商广告

#### 3.4.2 模板卡片网格

**布局**：响应式网格，默认 3-4 列

**单个卡片结构**：
```
┌────────────────────────┐
│                        │
│      缩略图            │
│                        │
├────────────────────────┤
│  模板名称              │
│               [使用]   │
└────────────────────────┘
```

**卡片元素**：

| 元素 | 描述 |
|------|------|
| 缩略图 | 16:9 或 1:1 比例，视频封面帧 |
| 模板名称 | 底部标题 |
| "使用"按钮 | 卡片右下角，主按钮样式 |

**交互状态**：

| 状态 | 视觉表现 |
|------|----------|
| 默认 | 正常显示 |
| Hover | 缩放/阴影提升 + 按钮高亮 |
| 选中 | 边框高亮 |

#### 3.4.3 底部操作区

**高级设置折叠面板**：
- 点击展开 → 显示更多参数（分辨率、时长、帧率等）
- 默认折叠状态

**自定义模板 ID 输入框**：
- 输入框 + 确认按钮
- 供高级用户手动输入模板

### 7.2 分类切换逻辑

```
用户点击分类Tab
    ↓
重置模板列表（显示加载状态，可选）
    ↓
根据分类ID过滤模板数据
    ↓
渲染新的模板卡片网格
```

### 7.3 模板选择逻辑

```
用户点击模板卡片 "使用" 按钮
    ↓
获取模板 ID 和模板类型
    ↓
如果有必填参数未填写 → 提示用户
    ↓
调用视频生成 API
    ↓
跳转到历史/结果页面
```

## 十一、注意事项

1. **API Key 安全**：不要在前端硬编码 API Key，建议通过环境变量或后端代理
2. **模板素材**：初期可用占位图，真实缩略图需要图片资源
3. **响应式**：移动端需要适配边栏收起/抽屉式导航
4. **Loading 状态**：API 调用时需要全局 Loading 或骨架屏
5. **错误处理**：网络失败、API 限流等场景需要友好提示
